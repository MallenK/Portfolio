Sobre mí

Sobre mí

Soy Sergi Mallén, desarrollador web full-stack. Construyo sitios rápidos, claros y fáciles de mantener. Me muevo cómodo en HTML, CSS y JavaScript, y trabajo backend con PHP y MySQL cuando el proyecto lo pide.

Disfruto convirtiendo ideas en productos: defino el alcance, maqueto, desarrollo y despliego con CI/CD. Me obsesionan dos cosas: rendimiento real (Core Web Vitals) y una UX que explique sin fricción.

He creado proyectos propios como un juego tipo GeoGuessr por España, un tracker de inversiones y una app de recetas. En freelance, ayudo a negocios locales a lanzar o mejorar su web con resultados medibles.

  • Stack: HTML/CSS/JS, PHP, MySQL, GitHub Actions.
  • Front: componentes accesibles, imágenes optimizadas, animaciones con control de movimiento.
  • Ops: Pages/Netlify, automatización de builds y fetchs de datos.
