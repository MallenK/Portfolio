Sobre mí

Soy Sergi Mallén, desarrollador web con más de cuatro años de experiencia. Me especializo en la creación de sitios corporativos y aplicaciones web funcionales, con una base sólida en el desarrollo Full-Stack.

Mi dominio abarca HTML, CSS, JavaScript, PHP y MySQL. Priorizo siempre un código limpio, una estructura lógica y la eficiencia del producto, lo que asegura el mantenimiento a largo plazo. Busco activamente nuevos retos técnicos para aplicar mis conocimientos y aportar valor.

He trabajado con *frameworks* como CodeIgniter y Symphony, además de tecnologías modernas como React y Node.js. Mi experiencia incluye la optimización SEO para mejorar la visibilidad online y el uso de API Rest en soluciones integrales.

  • Stack Principal: PHP, MySQL , CodeIgniter y React/Node.js.
  • Filosofía: Código limpio, eficiencia y enfoque en la fiabilidad del Backend.
  • Herramientas clave: Git, JQuery, WordPress, Figma y SEO.
